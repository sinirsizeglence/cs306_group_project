# Project Details

## Description

This project will focus on HIV/AIDS datasets in five weak entities.
Weak entities:

- Incidents
- Deaths
- Prevalence
- ART(treatment)
- Life Expectancy

The objective we aimed for this project is to design a medical database application and be a relational storage for such data.
